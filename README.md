#  Bookcase

Уеб приложение за управление на библиотечна система, реализирано чрез
REST API и отделен frontend клиент.

------------------------------------------------------------------------

##  Описание на проекта

**Bookcase** представлява клиент-сървърно уеб приложение, което
позволява управление на записи чрез стандартни CRUD операции:

-    Добавяне на запис
-    Преглед на всички запис
-    Редактиране на съществуваща запис
-    Изтриване на запис

Проектът е разделен на две основни части:

-   **BookcaseAPI** -- backend (REST API)
-   **Frontend приложение** -- клиентска част за визуализация и
    управление на данните

Архитектурата следва принципите на REST и Separation of Concerns.

------------------------------------------------------------------------

##  Архитектура

Frontend (Client)\
↓ HTTP Requests\
REST API (Backend)\
↓\
Database

------------------------------------------------------------------------

##  Използвани технологии

### Backend

-   Node.js (или съответната сървърна среда)
-   REST API
-   База данни
-   npm за управление на зависимости

### Frontend

-   JavaScript / TypeScript
-   HTML5
-   CSS
-   Fetch / Axios (за HTTP заявки)

------------------------------------------------------------------------

##  API Endpoints

  Метод    Endpoint     Описание
  -------- ------------ --------------------
  GET      /books       Връща всички записи
  GET      /books/:id   Връща запис по ID
  POST     /books       Добавя нов запис
  PUT      /books/:id   Редактира запис
  DELETE   /books/:id   Изтрива запис
